### Install dependencies:

```bash
yarn
```

### Start dev mode:

```bash
yarn start
```

### Build project:

```bash
yarn build
```

### Build project in ZIP:

```bash
yarn build:zip
```

### ESLint script fix:

```bash
yarn lint:script
```

### Prettier code write:

```bash
yarn prettier:write
```

### Create component:

```bash
yarn component:create <Name Component> <Path from src/component/>
```

### Create remove:

```bash
yarn component:remove <Name Component> <Path from src/component/>
```
